🌐 Hydronix Web System - User Journey Mechanism

1. 🔐 Login Stage

The user enters the website and the system checks authentication using browser storage.

localStorage.getItem("loggedIn")

Logic:

  • If NOT logged in → redirect to login page
  • If logged in → open dashboard

2. 🏠 Dashboard Loading

After login, the system loads all UI components automatically.

  • Sidebar navigation
  • Weather cards
  • Chart visualization
  • Map section
  • Language selector

3. 👤 Personalization

The system retrieves the username and shows a welcome message.

localStorage.getItem("username")

This improves user experience by making the system personalized.

4. 🌧️ Weather Data Loading

The system automatically fetches real-time weather data from OpenWeatherMap API.

fetch("https://api.openweathermap.org/data/2.5/forecast")
  • Rain data is extracted
  • Data is processed into arrays
  • Displayed in chart & UI cards

5. 📊 Data Visualization

Data is displayed using Chart.js for better understanding.

  • Line chart for rainfall
  • Status indicators (Low / Moderate / High)

6. 🗺️ Map Interaction

The system requests user location and displays it on an interactive map.

navigator.geolocation
  • Gets latitude & longitude
  • Loads Leaflet map
  • Adds marker with weather info

7. 🤖 Prediction Feature

When user clicks "Predict Rain", the system analyzes recent data.

  • Collects last weather values
  • Calculates average rainfall
  • Classifies risk level

Output: Low / Moderate / High risk

8. 🌍 Language System

User can switch between languages dynamically.

  • English
  • Arabic
  • French

Selected language is saved using localStorage.

9. 💬 Notification System

A welcome toast message appears after login.

  • Shows username
  • Disappears automatically after 3 seconds

🔄 Full System Flow

User opens website ↓ Login check (localStorage) ↓ Dashboard loads ↓ Weather API fetch ↓ Process data ↓ Render chart + map ↓ User interaction (predict / language) ↓ Dynamic UI updates

🏆 Summary

The system provides a complete interactive web experience combining authentication, real-time weather data, geolocation mapping, visualization, and predictive analytics inside a single dynamic dashboard.
